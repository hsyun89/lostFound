# lostfound

### 시작 

1. 처음 시작의 경우 

   $ git clone https://github.com/hsyun89/lostFound.git

2. 이후 이름명_기능명으로 브랜치 만들어 주세요

   $ git checkout -b 브랜치명(ex. hyunsu_login)

3. 브랜치 삭제는

   $ git branch -d 브랜치명

### 매일

##### 작업 전

1. 매일 프로젝트 전에는 풀을 땡겨주세요

   $ git pull

##### 작업 후 

1. 깃 애드

   $ git add .

2. 깃 커밋 

   $ git commit -m '커밋내용'

3. 푸시

   $ git push origin 브랜치명
   
4. 홍홍
